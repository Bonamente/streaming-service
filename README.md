# Paradiso
A streaming service with passwordless authentication, a favorite videos page, and a previously watched videos section.

## Technologies
- **JavaScript**
- **Next.js**
- **Magic Auth**
- **Hasura GraphQL Engine**
- **YouTube Data API**
- **Vercel**
- **jsonwebtoken**
- **jose** 
- **cookie**
- **classnames**
